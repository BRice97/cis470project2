Key Facts & Importance

Summary

What Is It 3D Printing turns 3D models into solid objects by building them in layers.

How It Works 3D printers print like blueprints. To create a 3D object, you have to start with a virtual design of the object through a CAD file. A CAD file is created using a 3D modeling application or a 3D scanner.

History In 1983, Chuck Hall created the 3D printing process called stereolithography. Therefore, despite 3D printing being a "futuristic" concept, it has been around for more than 30 years. 3D printing received all of its attention in 2009. Before 2009, 3D printing was mostly limited to industrial use, but through a project with RepRap, the first 3D printer was born. This computer was able to be create because patents were starting to expire.
